Piper Partner2021で使用するLab11 (CI/CD)用サンプルアプリケーションおよび関連ファイルです。  

このLabでは、CI/CDにJenkinsではなくAWS CodeDeploy/CodePipelineを使用します。

AWSでの構成時に、このテキストファイルから内容をcopy&pasteします。
